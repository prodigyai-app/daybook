      Connect Google Calendar (one-time setup)

      Live sync needs the app to run from a web address, not a double-clicked file. Serve this file from localhost or host it, then:

      1. Go to the Google Cloud Console → create a project → enable the Google Calendar API.
      2. Under Credentials, create an OAuth 2.0 Client ID (type: Web application). Add your address (e.g. http://localhost:8000) to Authorized JavaScript origins.
      3. Add yourself as a Test user on the OAuth consent screen.
      4. Paste the Client ID below and press Connect. Your ID is stored only in this browser.
